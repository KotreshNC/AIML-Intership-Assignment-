# AIML Internship Assignment

## AI Around Me

1. **Smartphone keyboard autocorrect/prediction** - uses language models (n-gram, RNN, or transformer) trained on text corpora to predict next words and correct typos.
2. **Email spam filtering** - uses classifiers (logistic regression / naive Bayes / tree models) trained on labeled emails to score and filter spam.
3. **Voice assistants (Siri/Google Assistant)** - use speech recognition models and intent classification to turn audio into text and actions.
4. **Recommendation systems (YouTube/Netflix)** - use collaborative filtering and deep learning on user/item interactions to predict content you’ll like.
5. **Navigation ETA (Google Maps)** - uses regression models on historical traffic and real-time sensor data to estimate arrival times.
6. **Bank fraud detection** - uses anomaly detection and classification on transaction patterns to flag suspicious activities.
7. **Face unlock** - uses CNNs for face detection and recognition to match a user’s face with stored biometric templates.
8. **Photo auto-enhancement** - uses neural image processing models to adjust brightness, contrast, and color automatically.
9. **Fitness tracker sleep stage detection** - uses time-series classification on heart rate and motion to infer sleep phases.
10. **Social feed ranking** - uses ranking and recommendation models to prioritize posts based on predicted engagement.
